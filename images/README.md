## 截图要求

### 格式要求
- 静态截图：PNG 格式（支持透明背景）
- 动态演示：GIF 格式（如需要展示操作过程）
- 分辨率：建议 1920x1080 或更高
- 文件大小：单个文件不超过 2MB

### 截图规范
1. **清晰度**：确保文字和界面元素清晰可见
2. **标注**：重要元素可用箭头或方框标注
3. **隐私**：敏感信息（如真实账号密码）需打码处理
4. **一致性**：保持截图风格一致，使用相同的浏览器主题
5. **完整性**：确保截图包含完整的操作界面

### 命名规范
- 使用英文和连字符
- 全部小写
- 描述性命名，如：`add-environment.png`
- 避免使用空格和特殊字符

## 截图制作建议

### 工具推荐
- **截图工具**：macOS 自带截图工具、Snagit、Lightshot
- **标注工具**：Skitch、Annotate、Snagit
- **GIF 制作**：LICEcap、Kap、ScreenToGif

### 制作步骤
1. 打开插件界面或操作页面
2. 调整浏览器窗口大小（建议 1920x1080）
3. 使用截图工具捕获界面
4. 使用标注工具添加说明（如需要）
5. 保存为 PNG 格式
6. 将文件放入本目录

## 注意事项

1. 所有截图文件应放在 `images/` 目录下
2. 文档中使用相对路径引用截图：`images/filename.png`
3. 截图更新后，需要同步更新文档中的说明文字
4. 确保截图文件名与文档中的引用一致
